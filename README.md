# cse111-Bloco-3-Python
O projeto consiste em:  Organização do código em funções. Uso de dicionários para armazenar dados. Entrada e saída de dados pelo usuário. Cálculo da média das notas. Verificação automática da aprovação do aluno. Geração de um arquivo resultado.txt com os resultados. Testes automatizados utilizando pytest para validar as funções principais.
